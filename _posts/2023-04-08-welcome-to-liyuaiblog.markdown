本文將指導您如何使用 Jekyll 部署一個靜態部落格網站。
首先，您需要在您的電腦上安裝 Jekyll。請遵循 [Jekyll 官方文檔](https://jekyllrb.com/docs/installation/) 中的說明進行操作。

在終端中運行以下命令，創建一個名為 `my-blog` 的新 Jekyll 部落格：
{% highlight ruby %}
jekyll new my-blog
{% endhighlight %}

{% highlight ruby %}
cd my-blog
{% endhighlight %}

{% highlight ruby %}
bundle exec jekyll serve
{% endhighlight %}
